# Setup instructions
This project uses python 2.7

# Install python deps:
```
pip install contextlib
pip install nltk
pip install numpy
pip install sklearn
pip install snowballstemmer

```

# Install nltk deps:
```
# python
> import nltk
> nltk.download()
# download the stopwords package
```
